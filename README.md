🧠 Overview

Designed a system-level UX improvement for a car leasing platform, focusing on reducing cognitive load and improving decision-making efficiency for different user types.

🎯 Problem

The original system suffered from:

High visual clutter and lack of hierarchy

Inefficient search process (no price filtering)

Difficulty in comparing options

Poor user flow for both private and business users

These issues impacted usability and slowed down decision-making.

🔍 Thinking Process

The approach included:

UX analysis using Gestalt principles (hierarchy, proximity, balance)

Heuristic evaluation based on Norman’s principles

Identification of key pain points across different user types

Mapping user journeys to understand real usage scenarios

Key insight:
👉 The main issue was not just visual design, but lack of structured system flow and decision support

👥 Users & Scenarios

Defined multiple personas with different needs:

Business user managing fleet decisions

Private user purchasing a family car

Occasional user renting for specific needs

Each persona highlighted different system requirements and constraints.

💡 Solution

Simplified UI with clear hierarchy and reduced visual noise

Introduced price filtering and structured data display

Designed consistent card/table layout for easy comparison

Improved navigation and system feedback

🧩 System Perspective ⭐

Focused on improving the system as a whole:

Structured data presentation (table-based comparison)

Clear separation between browsing, selection, and action

Defined constraints (e.g., required user details before booking)

Improved interaction between user actions and system feedback

👉 The solution treats UX as part of a decision-support system, not just interface design.

🎨 UX Decisions

Limited color palette to improve clarity and hierarchy

Emphasis on key actions (e.g., order button, vehicle data)

Consistent layout patterns to reduce cognitive load

📈 Improvements Identified

Add real-time vehicle availability

Improve system feedback and status tracking

Expand filtering and personalization options

Integrate better business-user workflows

📌 Tools & Methods

Design Sprint methodology

Heuristic Evaluation

User Journey Mapping

Persona Definition

UI Prototyping (Canva)

💬 Key Takeaway

This project focuses on system-level UX thinking, emphasizing structure, usability, and decision-making efficiency rather than visual design alone.
